# Synthetic Calibration Overlay Examples

These CSV files are synthetic calibration examples generated from the Ideal
Momentum Jet Explorer model. They are not measured data, not public validation
data, and not evidence that the reduced-order model is predictive.

Use them only for checking the data-overlay import interface and the
spreading-angle calibration workflow.

Each file uses normalized columns. The `zeta` column is normalized axial
distance `z / De`; the remaining columns are already nondimensional model
quantities such as `vhat`, `Ahat`, or `rhohat`.
